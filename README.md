# Comunicação-Entre-2VMs
Uma coneão entre as distros do Arch e o Debian via SSH

📌 Na prática isso serve para:
Acesso remoto → administrar um servidor via SSH
Transferência de arquivos → backups, deploy de sistemas
Testes de rede → IP, firewall, portas
Simular servidor + cliente
Estudar segurança (bloqueio/liberação de portas)

💻 Exemplo real:
Debian pode ser o servidor
Arch pode ser sua máquina cliente
Você acessa, envia arquivos, roda serviços
